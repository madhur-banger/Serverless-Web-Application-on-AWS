Project Overview

This project demonstrates the creation of a serverless web application using AWS services such as Lambda for logic, DynamoDB for storage, and S3 for hosting. The architecture eliminates the need for traditional server management.

Architecture Overview

The core AWS services include:

  • AWS Lambda: Serverless backend logic for handling CRUD operations on DynamoDB.
  • DynamoDB: A NoSQL database to manage application data.
  • S3: Static web hosting for the frontend.
  • CloudFront: CDN for secure and low-latency delivery of static content.

Project Workflow

  • Create a DynamoDB table for data storage.
  • Develop Lambda functions to handle CRUD operations on the database.
  • Host the frontend in S3 and use CloudFront for delivery.

Expected Outcomes

  • Gain hands-on experience with serverless technologies.
  • Efficient content delivery using CloudFront.
  • Scalable architecture with no server management.

Key Learning Areas

  • Serverless architecture design and scalability.
  • Lambda functions, DynamoDB, and S3 integration.
  • Optimizing content delivery with CloudFront.
