Welcome to the Project Documentation

Explore the architecture and learn how we built a serverless web app using AWS Lambda, DynamoDB, and S3.

Project Overview

This project demonstrates a serverless architecture using AWS services like Lambda, DynamoDB, and S3.

Architecture Overview

The key AWS services used include Lambda for backend logic, DynamoDB for data storage, and S3 for frontend hosting.

Project Workflow

  • Create a DynamoDB table
  • Develop Lambda functions for CRUD operations
  • Host frontend assets in S3
  • Set up CloudFront for content delivery

Expected Outcomes

  • Serverless app leveraging AWS services
  • Hands-on experience with Lambda and DynamoDB
  • Optimized content delivery using CloudFront

Key Learning Areas

  • Serverless architecture design
  • Lambda functions and API Gateway integration
  • NoSQL databases with DynamoDB
  • S3 & CloudFront for fast, secure delivery
